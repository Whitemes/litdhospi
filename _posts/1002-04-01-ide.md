## IDEs

Développement Java et Spring Boot. Autocomplétion intelligente, refactoring puissant et des outils de débogage avancés.  
*=> Référence en entreprise pour le développement Java.*

### **VS Code : Éditeur de code front**

Développement Web (Next.js/TypeScript).  
*=> Choisi pour sa rapidité et parce qu'il domine le marché du développement frontend.*
