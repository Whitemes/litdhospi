## Documentation Technique

### **Javadoc : Doc Java**

Produit des pages HTML d'API à partir des commentaires du code source Java.  
*=> Outil universel pour documenter du code Java.*

### **TypeDoc : Doc TypeScript**

Convertit les commentaires du code en un site de documentation statique pour les projets TypeScript.  
*=> Equivalent de Javadoc pour TypeScript et s'intègre parfaitement aux types forts du langage.*
