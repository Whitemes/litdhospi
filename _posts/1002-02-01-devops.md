## Versioning & DevOps

### **GitLab : Gestion de versions de code**

Centraliser le code source, gérer les différentes versions via les branches, revues de code avant fusion.  
*=> Intégré avec des outils de CI/CD.*

**GitLab CI/CD :**
Intégration et déploiement continus. Compilation automatique, exécution des tests et déploiement de l'application à chaque modif validée.  
*=> Intégré à GitLab et se configure simplement via un fichier `.gitlab-ci.yml`.*
